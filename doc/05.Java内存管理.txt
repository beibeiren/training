Java内存管理
	Java内存管理
		Java内存管理原理
			程序 <----> JVM <----> 操作系统
			Java程序，Java虚拟机，操作系统之间的关系
		方法区：存放类结构和main方法
		栈：1M空间，只存放内存的内存地址和基本数据类型
		堆：存放具体的数值
	Java数据类型
		基本数据类型：只在栈中分配一块内存
	    引用数据类型：在栈和堆中各分配一块内存
	    数组，接口：在栈和堆中各分配一块内存
	声明创建对象
	    声明对象：在栈中声明指针变量，他的内存不存放具体的数值，而是存放另一块堆中的内存地址
	    	String str; 
			栈         堆    方法区
			str
	    创建对象：在堆中分配一块存放具体值的内存，并把这个内存地址付给声明的指针变量
	  		str = new String（"abc"）        
			栈         堆    方法区
			str        "abc"
  	值传递与引用传递
    	按值传递参数数值不变，按引用传递参数数值改变
    	基本类型值传递，引用类型，数组引用传递
	特殊的String类
		String长度不可变,当String类型的变量的值改变时，JVM会重新给其分配一块新的内存空间
	无法改变对象
